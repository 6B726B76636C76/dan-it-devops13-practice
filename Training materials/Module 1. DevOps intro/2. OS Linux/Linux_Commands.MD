# 1. Navigation Commands

### `pwd`
Show the current working directory.

### `ls`
List files and directories.  
Common flags:
- `ls -l` ‚Äì long format  
- `ls -a` ‚Äì include hidden files  
- `ls -lh` ‚Äì human-readable sizes  

### `cd`
Change directory.
- `cd ..` ‚Äì go up one level  
- `cd` ‚Äì go to home directory  

---

# 2. File & Directory Operations

### `touch <file>`
Create an empty file or update its timestamp.

### `mkdir <directory>`
Create a directory.  
- `mkdir -p path/to/dir` ‚Äì create nested directories

### `cp <source> <destination>`
Copy files.
- `cp -r dir1 dir2` ‚Äì copy a directory recursively

### `mv <source> <destination>`
Move or rename files or directories.

### `rm <file>`
Remove files.  
‚ö†Ô∏è Be careful with:
- `rm -r dir` ‚Äì remove directory  
- `rm -rf /` ‚Äì **never run**, destructive

---

# 3. Viewing File Contents

### `cat <file>`
Print entire file.

### `less <file>`
Scroll through file (press `q` to quit).

### `head <file>`
Show first lines (default: 10).

### `tail <file>`
Show last lines.  
- `tail -f file` ‚Äì follow file in real time

---

# 4. Searching & Filtering

### `grep "pattern" <file>`
Search for text inside files.  
- `grep -r "text" .` ‚Äì recursive search  
- `grep -i` ‚Äì ignore case  

### `find <path> -name "*.log"`
Find files by name, type, size, etc.

---

# 5. System Information

### `whoami`
Show current user.

### `uname -a`
Display system and kernel information.

### `df -h`
Disk usage overview.

### `du -sh <dir>`
Size of a directory.

### `free -h` (Linux only)
Show memory usage.

### `top` / `htop`
Real-time process monitor.

### `ps aux`
List all running processes.

---

# 6. Networking Commands

### `ping <host>`
Check connectivity.

### `curl <url>`
Make HTTP requests from CLI.

### `ssh user@host`
Connect to remote server via SSH.

### `scp <file> user@host:/path`
Copy files over SSH.

### `ifconfig` / `ip addr`
View network interfaces.

---

# 7. Archiving & Compression

### `tar`
- `tar -czvf archive.tar.gz dir/` ‚Äì create  
- `tar -xzvf archive.tar.gz` ‚Äì extract

### `zip` / `unzip`
Work with ZIP archives.

---

# 8. Permissions & Ownership

### `chmod`
Change permissions.  
Examples:
- `chmod 755 file`
- `chmod u+x script.sh`

### `chown user:group file`
Change file owner.

---

# 9. Package Managers Overview

### Ubuntu / Debian
```
sudo apt update
sudo apt install <package>
sudo apt remove <package>
```

### CentOS / RHEL
```
sudo dnf install <package>
sudo yum install <package>
```

### macOS (Homebrew)
```
brew install <package>
brew update
brew remove <package>
```
